# ✨ Generative AI

Build with large language models, image models, and the tools of the generative era.

## Understand it

- [Elements of AI](https://www.elementsofai.com/) — a free, non-technical intro to what AI is and isn't.
- [DeepLearning.AI Short Courses](https://www.deeplearning.ai/short-courses/) — free, focused courses on LLMs, RAG, agents, and more.
- [Generative AI for Beginners (Microsoft)](https://microsoft.github.io/generative-ai-for-beginners/) — a free 21-lesson course, hands-on.

## Build with LLMs

- [Anthropic Docs](https://docs.anthropic.com/) — build AI applications with Claude.
- [Hugging Face Learn](https://huggingface.co/learn) — open-source models, NLP, and LLMs, hands-on.
- [OpenAI Cookbook](https://github.com/openai/openai-cookbook) — practical code examples for building with LLMs.
- [LangChain Docs](https://python.langchain.com/docs/introduction/) — a popular framework for building LLM applications.

## Prompt well

- [Prompt Engineering Guide](https://www.promptingguide.ai/) — a thorough, vendor-neutral guide to prompting.
- [Anthropic Prompt Engineering](https://docs.anthropic.com/en/docs/build-with-claude/prompt-engineering/overview) — practical prompting techniques.

## Generative tools

- [Awesome Generative AI](https://github.com/steven2358/awesome-generative-ai) — a curated list of generative AI tools and projects.
