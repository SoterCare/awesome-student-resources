# 🧠 Machine Learning

The fundamentals of ML — the math, models, and frameworks behind modern AI.

## Courses

- [Machine Learning Specialization (Andrew Ng)](https://www.coursera.org/specializations/machine-learning-introduction) — the classic foundation; audit for free.
- [Google Machine Learning Crash Course](https://developers.google.com/machine-learning/crash-course) — practical, visual, and free.
- [Kaggle Learn](https://www.kaggle.com/learn) — bite-sized hands-on courses with instant feedback.
- [fast.ai — Practical Deep Learning](https://course.fast.ai/) — build working models first, theory second.

## Intuition

- [StatQuest (Josh Starmer)](https://statquest.org/) — the clearest explanations of ML and statistics concepts anywhere.
- [3Blue1Brown — Neural Networks](https://www.3blue1brown.com/topics/neural-networks) — beautiful visual intuition for how neural nets learn.

## Books & practice

- [Dive into Deep Learning (d2l.ai)](https://d2l.ai/) — a free interactive book with code, math, and discussions.
- [Made With ML](https://madewithml.com/) — how to build and ship production ML, free.
- [scikit-learn Documentation](https://scikit-learn.org/stable/) — the reference for classical ML in Python.
