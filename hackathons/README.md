# 🏆 Hackathons

Find hackathons, learn to win them, and eventually run your own.

## Find hackathons

- [MLH — Season Events](https://mlh.io/seasons) — the global calendar of student hackathons.
- [Devpost](https://devpost.com/hackathons) — find and enter online and in-person hackathons.
- [Devfolio](https://devfolio.co/hackathons) — a large hackathon platform, popular in South Asia.

## Win hackathons

- [Awesome Hackathon](https://github.com/dribdat/awesome-hackathon) — resources for participating in and running hackathons.
- [MLH Organizer Guide](https://guide.mlh.io/) — understand how hackathons are run and judged; knowing this helps you win them.

## Run your own

- SoterCare has run [VisioNEX](https://github.com/SoterCare/community/tree/main/hackathons/visionex) — reaching 500+ students across 40+ schools.
- [How to Organize a Hackathon (SoterCare)](https://github.com/SoterCare/community/blob/main/guides/organize-a-hackathon.md) — our full organizing playbook.
