# 🌐 Web Development

Everything for building on the web — frontend, backend, and full-stack.

## Full curricula

- [The Odin Project](https://www.theodinproject.com/) — a complete, free, project-driven full-stack curriculum.
- [Full Stack Open (University of Helsinki)](https://fullstackopen.com/en/) — modern React, Node, GraphQL, and TypeScript, free.
- [freeCodeCamp — Responsive Web Design](https://www.freecodecamp.org/learn/2022/responsive-web-design/) — HTML & CSS from scratch, with a certification.

## Reference & best practices

- [MDN Web Docs](https://developer.mozilla.org/) — the definitive reference for HTML, CSS, and JavaScript.
- [web.dev (Google)](https://web.dev/learn/) — modern web best practices: performance, accessibility, and more.

## Practice & roadmaps

- [Frontend Mentor](https://www.frontendmentor.io/) — real-world frontend challenges with designs provided.
- [roadmap.sh — Frontend](https://roadmap.sh/frontend) & [Backend](https://roadmap.sh/backend) — visual step-by-step learning paths.

## Frameworks (official docs are the best start)

- [React](https://react.dev/learn) · [Next.js](https://nextjs.org/learn) · [Vue](https://vuejs.org/tutorial/) · [Node.js](https://nodejs.org/en/learn)
