# 🟧 AWS

Amazon Web Services — the largest cloud platform, with a generous student program.

## Start here

- [AWS Educate](https://aws.amazon.com/education/awseducate/) — free credits, labs, and courses for students, no credit card.
- [AWS Cloud Practitioner Essentials](https://explore.skillbuilder.aws/learn/course/134/aws-cloud-practitioner-essentials) — the standard first course.

## Go deeper

- [AWS Skill Builder](https://skillbuilder.aws/) — AWS's official platform, hundreds of free courses.
- [AWS Free Tier](https://aws.amazon.com/free/) — build real projects within the always-free limits.
- [AWS Well-Architected Framework](https://aws.amazon.com/architecture/well-architected/) — how to design cloud systems well.

## Certifications worth targeting

- **Cloud Practitioner** — foundational, great first cert.
- **Solutions Architect – Associate** — the most recognized associate-level cert.

## Community

Connect through an [AWS Cloud Club](https://aws.amazon.com/developer/community/cloud-clubs/) — SoterCare engages with this community.
