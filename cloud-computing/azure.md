# 🔵 Microsoft Azure

Microsoft's cloud — strong in enterprise, and well-integrated with the Microsoft ecosystem.

## Start here

- [Azure for Students](https://azure.microsoft.com/en-us/free/students/) — free credit and services, no credit card required.
- [Microsoft Learn — Azure](https://learn.microsoft.com/en-us/training/azure/) — free, official, hands-on learning paths.

## Go deeper

- [AZ-900: Azure Fundamentals](https://learn.microsoft.com/en-us/credentials/certifications/azure-fundamentals/) — the foundational certification path.
- [Microsoft Learn — Cloud Skills Challenge](https://learn.microsoft.com/en-us/collections/) — guided, gamified learning collections.

## Certifications worth targeting

- **AZ-900 (Fundamentals)** — the best first Azure cert.
- **AZ-104 (Administrator)** — a strong next step.
