# 🔴 Google Cloud (GCP)

Google's cloud — strong in data, analytics, and machine learning.

## Start here

- [Google Cloud Skills Boost](https://www.cloudskillsboost.google/) — hands-on labs and quests in real GCP environments.
- [Google Cloud Free Tier](https://cloud.google.com/free) — free credits plus always-free products.

## Go deeper

- [Google Cloud Documentation](https://cloud.google.com/docs) — official docs and quickstarts.
- [Google Cloud — Training](https://cloud.google.com/learn/training) — role-based learning paths and certifications.

## Certifications worth targeting

- **Cloud Digital Leader** — foundational, non-technical entry point.
- **Associate Cloud Engineer** — the standard hands-on associate cert.
