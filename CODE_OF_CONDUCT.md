# Code of Conduct

This project follows the SoterCare community Code of Conduct.

We expect everyone to:

- Be respectful and welcoming
- Assume good intent
- Give and accept constructive feedback graciously
- Keep contributions relevant and honest (no spam, no affiliate links)

Harassment, discrimination, or abusive behavior will not be tolerated.

Concerns can be reported to the SoterCare core team via [LinkedIn](https://www.linkedin.com/company/sotercare) or [sotercare.com](https://sotercare.com). The full community Code of Conduct lives in the [community repository](https://github.com/SoterCare/community/blob/main/CODE_OF_CONDUCT.md).

Adapted from the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.
