# 🔧 Git Guide

Git is the version-control system underneath GitHub. Learn it once, use it forever.

## Learn

- [Learn Git Branching](https://learngitbranching.js.org/) — visual, game-like, the fastest way to understand branching.
- [Pro Git book](https://git-scm.com/book/en/v2) — free and definitive; chapters 1–3 cover what most people need.
- [Atlassian Git Tutorials](https://www.atlassian.com/git/tutorials) — clear, well-organized explanations of every command.

## When things go wrong

- [Oh Shit, Git!?!](https://ohshitgit.com/) — how to get out of common Git messes.
- [Dangit, Git!?!](https://dangitgit.com/) — the same, in polite language.

## Reference

- [Git Documentation](https://git-scm.com/doc) — the official manual.
- [GitHub Git Cheat Sheet (PDF)](https://education.github.com/git-cheat-sheet-education.pdf) — one page of the commands you'll actually use.
