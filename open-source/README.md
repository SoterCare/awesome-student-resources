## Find things to work on

- [Good First Issue](https://goodfirstissue.dev/) — curated beginner-friendly issues.
- [Up For Grabs](https://up-for-grabs.net/) — projects with tasks set aside for newcomers.
- [good first issues across SoterCare](https://github.com/search?q=org%3ASoterCare+label%3A%22good+first+issue%22+state%3Aopen&type=issues) — contribute to our own projects.

## Why it matters

Open source teaches you to read code, collaborate, take feedback, and ship — the exact skills employers look for, and every contribution is portfolio evidence.
