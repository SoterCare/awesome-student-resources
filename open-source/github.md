# 🐙 Getting Started with Git & GitHub

Git is the version-control system; GitHub is where the world collaborates with it. Learn both here.

## Git — the fundamentals

- [Learn Git Branching](https://learngitbranching.js.org/) — visual, game-like, the fastest way to understand branching.
- [Pro Git book](https://git-scm.com/book/en/v2) — free and definitive; chapters 1–3 cover what most people need.
- [Atlassian Git Tutorials](https://www.atlassian.com/git/tutorials) — clear, well-organized explanations of every command.
- [Oh Shit, Git!?!](https://ohshitgit.com/) — how to get out of common Git messes.

## GitHub — collaborate

- [GitHub Skills](https://skills.github.com/) — interactive courses that run on GitHub itself; learn by doing in minutes.
- [GitHub Docs](https://docs.github.com/) — the official manual for everything GitHub.
- [Understanding the GitHub Flow](https://docs.github.com/en/get-started/using-github/github-flow) — the standard branch-and-PR workflow.
- [GitHub CLI](https://cli.github.com/) — do GitHub from your terminal.

## Beginner-friendly (SoterCare)

- [GitHub Guide for Beginners](https://github.com/SoterCare/community/blob/main/guides/github-for-beginners.md) — our own zero-to-first-PR walkthrough.

## Reference

- [GitHub Git Cheat Sheet (PDF)](https://education.github.com/git-cheat-sheet-education.pdf) — one page of the commands you'll actually use.
