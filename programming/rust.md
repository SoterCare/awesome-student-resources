# 🦀 Rust

A modern systems language focused on performance and safety — and consistently the "most loved" language in developer surveys.

## Learn

- [The Rust Programming Language ("the Book")](https://doc.rust-lang.org/book/) — the official, free, and excellent starting point.
- [Rustlings](https://github.com/rust-lang/rustlings) — small exercises to learn Rust by fixing code.
- [Rust by Example](https://doc.rust-lang.org/rust-by-example/) — learn through runnable examples.
- [Comprehensive Rust (Google)](https://google.github.io/comprehensive-rust/) — Google's free multi-day Rust course.

## Reference

- [Rust Standard Library Docs](https://doc.rust-lang.org/std/) — the official API reference.
- [The Cargo Book](https://doc.rust-lang.org/cargo/) — Rust's build tool and package manager.

## Practice

- [Exercism — Rust Track](https://exercism.org/tracks/rust) — free exercises with mentoring.
