# 🐍 Python

The friendliest first language, and a workhorse for AI, data, web, and automation.

## Learn

- [Python for Everybody (py4e)](https://www.py4e.com/) — the gold-standard free intro course, by Dr. Chuck.
- [Automate the Boring Stuff](https://automatetheboringstuff.com/) — learn Python by automating real tasks; free online.
- [CS50's Introduction to Python](https://cs50.harvard.edu/python/) — Harvard's rigorous, free Python course.
- [Real Python](https://realpython.com/) — deep, practical tutorials on nearly every Python topic.

## Reference

- [Official Python Docs & Tutorial](https://docs.python.org/3/tutorial/) — the canonical reference.
- [Python Cheatsheet](https://www.pythoncheatsheet.org/) — quick syntax lookups.

## Practice

- [Exercism — Python Track](https://exercism.org/tracks/python) — free exercises with mentor feedback.
- [HackerRank — Python](https://www.hackerrank.com/domains/python) — bite-sized challenges.
