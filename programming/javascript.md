# 🟨 JavaScript

The language of the web — and increasingly of servers, apps, and tooling too.

## Learn

- [javascript.info](https://javascript.info/) — the modern JavaScript tutorial, thorough and free.
- [Eloquent JavaScript](https://eloquentjavascript.net/) — a beautifully written free book with in-browser exercises.
- [freeCodeCamp — JavaScript](https://www.freecodecamp.org/learn/javascript-algorithms-and-data-structures/) — hands-on curriculum with a certification.
- [JavaScript30](https://javascript30.com/) — 30 projects in 30 days, no frameworks, free.

## Reference

- [MDN — JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript) — the definitive web reference.
- [You Don't Know JS Yet](https://github.com/getify/You-Dont-Know-JS) — free book series that goes deep on the language.

## Practice

- [Exercism — JavaScript Track](https://exercism.org/tracks/javascript) — free exercises with mentoring.
