# Contributing

Thanks for helping make this list better! This is a community-curated resource — quality over quantity.

## What makes a good addition

Every entry must be:

- **Free-first** — free, or with a genuinely useful free tier
- **Student-relevant** — helps a student learn, contribute, or grow
- **Backed by a reason** — one line on *why* it's worth someone's time (not just what it is)
- **Currently maintained** — no dead links or abandoned resources

## Format

Add your entry to the right section, alphabetically friendly, in this format:

```markdown
- [Resource Name](https://link) — one-line reason it's worth someone's time.
```

Good:
> - [GitHub Skills](https://skills.github.com/) — interactive courses that run on GitHub itself; learn by doing in minutes.

Not enough:
> - [GitHub Skills](https://skills.github.com/) — courses about GitHub.

## How to contribute

1. Fork this repository
2. Add your entry to the relevant section of `README.md`
3. Check your link works and follows the format
4. Open a Pull Request describing what you added and why

## Suggesting a new section

Open an issue first so we can discuss where it fits.

## Standards

- One resource per line
- No affiliate or referral links
- No duplicates — search first
- English descriptions (translations welcome as separate entries)
