# 🎁 GitHub Student Developer Pack

Arguably the single best free offer for any student developer — dozens of professional tools and credits, free while you're a student.

## Get it

- [GitHub Student Developer Pack](https://education.github.com/pack) — apply with your student email or enrollment proof.

## What's inside (highlights)

- **GitHub Pro** — free while you're a student.
- **Free domain names** (Namecheap, .tech) and hosting credits.
- **Cloud credits** — DigitalOcean, Microsoft Azure, Heroku alternatives, and more.
- **Learning** — free access to courses and platforms (e.g. DataCamp, Educative).
- **Dev tools** — JetBrains IDEs, GitKraken, and many others, free.

## Tips

- Apply as early as possible — the credits are genuinely useful for real projects.
- Reapply/renew if your student status is re-verified.
- Combine with [GitHub Campus Experts](ambassador-programs.md) to level up further.
