# △ Algorand

A pure proof-of-stake blockchain. SoterCare ran an [Algorand Foundation Workshop](https://github.com/SoterCare/community/blob/main/events/algorand-workshop.md) (AlgoKit 3.0, 50+ students trained).

## Learn

- [Algorand Developer Portal](https://developer.algorand.org/) — official docs, tutorials, and guides.
- [AlgoKit](https://dev.algorand.co/algokit/algokit-intro/) — the toolkit we used in our workshop; the fastest way to start building.
- [Algorand Developer Docs](https://dev.algorand.co/) — the current developer documentation hub.

## Build

- [TEAL & AVM Guidelines](https://developer.algorand.org/docs/get-details/dapps/avm/teal/) — understand Algorand's smart-contract layer.
